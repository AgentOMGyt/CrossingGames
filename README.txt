# 🌳 Crossing Games

**Crossing Games** est le site vitrine officiel d'un serveur Minecraft communautaire. Conçu pour offrir une immersion immédiate, le site guide les joueurs à travers les mises à jour du projet, le système de vote et les étapes pour rejoindre l'aventure.

---

## ✨ Fonctionnalités du Site

- **Interface Immersive** : Un design chaleureux aux couleurs de la nature, avec des animations fluides (logo dynamique au scroll, mascotte animée).
- **Système de Mise à Jour** : Une page dédiée (`update.html`) listant l'historique des versions et les patchs appliqués au serveur.
- **Espace de Vote** : Une interface simplifiée pour encourager la communauté à soutenir le serveur sur différentes plateformes.
- **Trailer Intégré** : Présentation vidéo du projet directement sur la page d'accueil.
- **Optimisation Mobile** : Menu et mise en page adaptatifs pour une consultation sur smartphone.

---

## 🛠️ Stack Technique

- **Frontend** : HTML5 sémantique.
- **Stylisation** : CSS3 avancé avec animations `@keyframes` (notamment pour la mascotte "CrossiBird" et les reflets des boutons).
- **Interactivité** : JavaScript (Vanilla) pour la gestion du défilement dynamique, de l'opacité des bannières et de la réactivité de l'interface.
- **Typographie** : Utilisation de la police moderne **Poppins** via Google Fonts.

---

## 📂 Structure du Projet

- `Accueil/` : Contient `index.html`, les styles et la logique de la page principale.
- `Update/` : Journal des modifications du serveur (`update.html`).
- `Vote/` : Liens vers les plateformes de vote pour le serveur.
- `source/` : Assets graphiques (images de fond, logos, icônes) et vidéos.

---

## 🚀 Comment rejoindre le serveur ?

Le site indique la marche à suivre pour les nouveaux joueurs :
1. Lancer Minecraft en version **1.21**.
2. Ajouter un nouveau serveur.
3. Entrer l'adresse IP : `PLAY.CROSSING.FR`.
4. Accepter le pack de textures personnalisé pour une expérience optimale.
